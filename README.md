## _OPQR_이란?
 본 프로그램은 
1. QR코드를 인식을 하고  
2. QR코드에 연동이 된 url을 검사하여 건전한 사이트인지,
아니면 위험한 사이트인지 분별하여 
3. 사용자에게 알려준다.
